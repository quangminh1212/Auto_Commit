Đây là file mẫu để thử nghiệm Auto Commit.
Chúng ta sẽ sử dụng file này để tạo commit và kiểm tra chức năng của ứng dụng.

Một số tính năng của Auto Commit:
1. Tự động tạo commit message bằng Gemini API
2. Giao diện đồ họa dễ sử dụng
3. Hỗ trợ chọn repository
4. Cấu hình API key và các thông số khác
5. Xem thông tin diff và commit
6. Tự động push lên remote repository (tùy chọn)
